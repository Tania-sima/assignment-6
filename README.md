NAME: DIGITOOLS PLATFORM

DESCRIPTION: This is a react based platform to handle real time shopping cart. It features a responsive UI built with tailwind css and asynchronous data fetching to load information dynamically.

TECHNOLOGY: React.js, tailwind css, daisy UI, font awesome, JSON data flow, react-toastify.

FEATURES: 1.count card item, 
          2.onclick card will remove each card, onclick proceed to checkout will remove all card, 
          3.dynamic shopping cart, toast notification, add total price of added cart.
